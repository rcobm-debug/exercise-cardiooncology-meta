# Material NO incluido (recomendado mantener fuera de GitHub)

Para evitar problemas de copyright y confidencialidad, este repositorio **no** incluye:

- PDFs a texto completo (incluidos o excluidos).
- Capturas/imágenes extraídas directamente de artículos a texto completo.
- Correspondencia con autores o plantillas de contacto.
- Versiones del manuscrito.

Si necesitas conservar estos materiales, guárdalos localmente (p. ej., carpeta `private/`) y asegúrate de que estén excluidos por `.gitignore`.
