# Paquete público (apto para GitHub)

Este repositorio contiene únicamente material recomendable para un repositorio público:
- Código Python para reproducir metaanálisis (DL + HK) y forest plots.
- Datos derivados de extracción (CSV/XLSX) necesarios para reproducibilidad.
- Trazabilidad por etapas (E4–E9) en formatos tabulares, sin incluir textos completos.
- Figuras finales (PNG) usadas en el manuscrito (Figuras 1–7).

Excluido deliberadamente:
- PDFs a texto completo y cualquier contenido protegido (textos completos, capturas de artículos).
- Correspondencia/plantillas de contacto.
- Copias del manuscrito.
